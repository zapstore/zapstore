# Zapstore

The open app store powered by your social network

[https://zapstore.dev](https://zapstore.dev)

## Build from source

```bash
flutter pub get
flutter build apk --split-per-abi --debug
```

APK will be available at `build/app/outputs/flutter-apk`.

## Contributing
